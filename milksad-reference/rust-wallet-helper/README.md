# Wallet Helper Library

This Rust crate contains some code related to cryptocurrency wallet research.

## Status

This repository contains unmaintained and unstable code which is meant for other researchers. We give no functionality or security guarantees. This code should not be used in any production environment.

Feedback is welcome, but we're unlikely to implement any feature requests.
